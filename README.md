# WiFi-Smart-Roam
magisk模块，双频WiFi6网络优化工具，支持三大功能：①双条件漫游（当前AP RSSI&lt;-65dBm且扫描到更好AP RSSI>-50dBm时触发重连）；②始终保持5GHz频段（仅在连接2.4GHz时触发检测，检测到同SSID下5GHz信号>-70dBm立即漫游切换）；③激进蜂窝切换策略（蜂窝RSRP>-85dBm即作为候选网络）。配合Magisk永久ROOT运行，开机自启。
Dual-band WiFi 6 network optimization tool, supporting three main functions: ① Dual-condition roaming (trigger reconnection when the current AP RSSI <-65dBm and a better AP with RSSI > -50dBm is detected); ② Always maintain the 5GHz band (only triggered when connected to 2.4GHz, immediately switch to 5GHz if a signal > -70dBm is detected under the same SSID); ③ Aggressive cellular handoff strategy (cellular RSRP > -85dBm is considered a candidate network). Works with Magisk permanent ROOT and starts automatically on boot. Detection interval: 15s, cooldown: 60s, can be modified at the top of the configuration file code.
